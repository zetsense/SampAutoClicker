## ?? Описание

**SampAutoClicker** — это мощный и удобный скрипт для MoonLoader, предназначенный для автоматизации кликов в игре SA:MP. Скрипт позволяет настраивать интервалы между кликами, количество кликов, выбирать кнопку мыши и устанавливать таймеры для автоматического завершения работы.

## ? Возможности

- ??? Автоматические клики с настраиваемой частотой
- ?? Гибкая настройка интервалов между кликами (миллисекунды, секунды, минуты)
- ?? Настройка количества кликов
- ?? Установка таймера для автоматического завершения работы
- ?? Поддержка различных кнопок мыши
- ?? Удобный и интуитивно понятный интерфейс

## ?? Установка

1. Убедитесь, что у вас установлен [MoonLoader](https://www.blast.hk/threads/13305/) версии 026 или выше
2. Скачайте последнюю версию скрипта из [релизов](https://github.com/Zetsense/SampAutoClicker/releases)
3. Распакуйте архив в папку `moonloader` вашей игры
4. Убедитесь, что структура файлов выглядит следующим образом:
   ```
   moonloader/
   ??? SampAutoClicker/
   ?   ??? lib/
   ?   ?   ??? _imguifunc.lua
   ?   ?   ??? _main.lua
   ?   ?   ??? _render.lua
   ?   ??? storage/
   ?   ?   ??? render.json
   ?   ??? _init_.lua
   ```

## ?? Использование

1. Запустите игру с установленным MoonLoader
2. Нажмите клавишу **END** для открытия меню скрипта
3. Настройте параметры автокликера:
   - Выберите кнопку мыши для кликов
   - Установите интервал между кликами
   - Укажите количество кликов
   - При необходимости настройте таймер автоматического завершения
4. Нажмите кнопку "Активировать" для запуска автокликера
5. Наслаждайтесь автоматизацией!

## ?? Настройки

Все настройки автоматически сохраняются в файл `storage/render.json` и загружаются при следующем запуске скрипта.

### Основные параметры:

| Параметр | Описание |
|----------|----------|
| Кнопка клика | Выбор кнопки мыши для автоматических кликов |
| Интервал между кликами | Время между отдельными кликами (мс, сек, мин) |
| Количество кликов | Сколько кликов будет выполнено за один цикл |
| Таймер завершения | Время, через которое автокликер автоматически отключится |
